# Razor Syntax

Razor ist die Template-Sprache von Blazor. Sie erlaubt es, **C#-Code direkt im HTML** zu schreiben — erkennbar am `@`-Zeichen.

## Themen

- [Werte ausgeben](./01-werte-ausgeben.md)
- [Bedingungen & Schleifen](./02-bedingungen-schleifen.md)
- [C#-Blöcke & Direktiven](./03-csharp-bloecke-direktiven.md)

> **Das ist wichtig weil...** Razor ist das Bindeglied zwischen deinen C#-Daten und der HTML-Anzeige. Ohne Razor-Syntax kannst du keine dynamischen Inhalte in Blazor darstellen.
