# Farben & Schriften

## Farben

```css
.element {
    color: #333333;             /* Textfarbe als Hex */
    background-color: #f0f0f0; /* Hintergrund als Hex */
    color: rgb(51, 51, 51);    /* RGB-Wert */
    color: red;                 /* Farbname */
}
```

## Schriften

```css
body {
    font-family: Arial, sans-serif; /* Schriftart */
    font-size: 16px;                /* Schriftgröße */
    font-weight: bold;              /* Schriftgewicht */
    line-height: 1.5;               /* Zeilenabstand */
    text-align: center;             /* Ausrichtung */
}
```

> **Das ist wichtig weil...** Konsistente Farben und Schriften machen deine App professionell. Definiere Farben am besten zentral als CSS-Variablen: `--primary: #0078d4;`
