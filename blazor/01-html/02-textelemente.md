# Textelemente

Mit HTML-Textelementen strukturierst du Überschriften, Absätze und hervorgehobenen Text.

## Beispiel

```html
<h1>Hauptüberschrift</h1>
<h2>Unterüberschrift</h2>
<h3>Noch kleiner</h3>

<p>Das ist ein normaler Absatz mit <strong>fettem</strong> und <em>kursivem</em> Text.</p>

<span>Inline-Element — kein Zeilenumbruch</span>
```

| Tag | Bedeutung |
|-----|-----------|
| `<h1>` bis `<h6>` | Überschriften (1 = größte) |
| `<p>` | Absatz (Paragraph) |
| `<strong>` | Fett / wichtig |
| `<em>` | Kursiv / betont |
| `<span>` | Inline-Container ohne Formatierung |

> **Das ist wichtig weil...** Überschriften helfen Benutzern und Suchmaschinen, den Inhalt zu verstehen. Nutze `<h1>` nur einmal pro Seite.
